        - Dùng thẻ img:
            - Các thuộc tính thường dùng: src , alt 
                + src: URL (đường dẫn đến file ảnh)
                + alt : nội dung mô tả
            - URL: 2 loại đường dẫn
                + URL tuyệt đối:
                    [giao thức]://domain/path_(folder/file)
                    Ví dụ:
                    https://devmaster.edu.vn/images/logo.png

                + URL tương đối:
                    ++ vào trong một thư mục:
                        folderName/
                    ++ ra ngoài một thư mục:
                        ../
                        ../../
            - target: chỉ đích đến
